Gigabyte B860M DS3H WIFI6E UD LGA1851 Motherboard

Gigabyte B860M DS3H WIFI6E UD LGA1851 Motherboard specifications:

  • CPU Support:

    • LGA1851 socket: Intel® Core™ Ultra Processors
    • L3 cache varies with CPU
  • Chipset: Intel® B860 Express Chipset
  • Memory:
    • DDR5 DIMM sockets supporting up to 256 GB (64 GB single DIMM capacity)
    • Dual channel memory architecture
    • Supports speeds up to 9066 MT/s (O.C.)
    • Supports ECC Un-buffered DIMM (operate in non-ECC mode)
    • Supports non-ECC Un-buffered DIMM
    • Supports XMP memory modules

  • Onboard Graphics:

    • Integrated Intel® HD Graphics
    • 1 x HDMI port (HDMI 2.1, 4K@60Hz, HDCP 2.3)
    • 2 x DisplayPort (DisplayPort 2.1, 4K@144Hz)
    • Triple-display support
  • Audio:
    • Realtek® Audio CODEC
    • High Definition Audio
    • 2/4/5.1/7.1-channel support
  • Networking:
    • Realtek® 2.5GbE LAN chip
    • Intel® Wi-Fi 6E AX211 (WIFI a/b/g/n/ac/ax, Bluetooth 5.3)
    • Realtek® Wi-Fi 6E RTL8852CE (WIFI a/b/g/n/ac/ax, Bluetooth 5.3)
  • Expansion Slots:
    • CPU:
      • 1 x PCI Express x16 slot (PCIEX16) supporting PCIe 5.0 x16
    • Chipset:
      • 3 x PCI Express x16 slots (PCIEX1_1/2/3) supporting PCIe 4.0 x1

  • Storage Interfaces:

    • CPU:
      • 1 x M.2 connector (M2A_CPU) supporting PCIe 5.0 x4/x2 SSDs
    • Chipset:
      • 1 x M.2 connector (M2N_SB) supporting PCIe 4.0 x4/x2 SSDs
      • 4 x SATA 6Gb/s connectors
    • RAID 0, RAID 1, RAID 5, RAID 10 support for SATA storage devices
  • USB Ports:
    • Back Panel:
      • 1 x USB 3.2 Gen 2 Type-A port (red)
      • 2 x USB 3.2 Gen 1 ports
      • 3 x USB 2.0/1.1 ports
    • Internal Headers:
      • 1 x USB Type-C® (USB 3.2 Gen 1)
      • 4 x USB 3.2 Gen 1 ports
      • 4 x USB 2.0/1.1 ports

  • Internal I/O Connectors:

    • Power connectors (24-pin ATX, 8-pin ATX 12V)
    • Fan headers (CPU, system, water cooling pump)
    • RGB LED headers
    • SATA connectors, front panel headers
  • BIOS:
    • 256 Mbit flash
    • AMI UEFI BIOS
    • PnP 1.0a, DMI 2.7, WfM 2.0, SM BIOS 2.7, ACPI 5.0
  • Unique Features:
    • GIGABYTE Control Center (GCC)
    • Q-Flash, Q-Flash Plus, Smart Backup

  • Bundled Software:

    • Norton® Internet Security (OEM version)
    • LAN bandwidth management software
  • Operating System Support:
    • Windows 11 64-bit
    • Windows 10 64-bit
    • Intel® NPU driver supported on Windows 11 only
  • Form Factor:
    • Micro ATX; 24.4cm x 24.4cm
  • Supports Intel® Core™ Ultra processors (Series 2)
  • Hybrid 5+1+2 phases VRM solution
  • D5 Bionic Corsa for Infinite Memory Performance

  • AI Perfdrive : Provides optimal and customized BIOS preset profile for users

  • WIFI EZ-Plug : Quick and easy design for Wi-Fi antenna installation
  • EZ-Latch : PCIe x16 Slot with Quick Release Design
  • Friendly UI : Multi-Theme, AIO Fan Control, and Q-Flash Auto Scan in BIOS and SW
  • New  Power Monitor  in  HWinfo for real-time monitoring on CPU power phases

  • Ultra-Fast Storage : 2*M.2 slots, including 1* PCIe 5.0 x4

  • M.2 Thermal Guard : To Ensure M.2 SSD Performance
  • Fast Networking : 2.5GbE LAN & Wi-Fi 6E 802.11ax
  • Extended Connectivity : 2*DisplayPort, HDMI
